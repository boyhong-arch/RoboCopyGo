# ReboCopy Go+ 是什么？
Robocopy Go+ 是：一款基于微软Windows系统内置命令行工具Robocopy.exe开发的图形用户界面（GUI）软件。 它的主要目的是让普通用户也能轻松使用功能强大但操作复杂的Robocopy.

# ReboCopy Go+ 的特点
- 核心基础：它的底层核心是微软官方的Robocopy（强健文件复制），这意味着它继承了原生工具高效、稳定的特性，例如支持多线程复制、断点续传、保留文件属性等。

- 界面优化：软件为Robocopy的命令行参数提供了直观的勾选界面和右键菜单，大大降低了使用门槛。

- 特色功能：除了基本的复制功能外，它还提供了一些便利功能，如“作业完成后关机、待机、休眠”、“外壳扩展添加到文件夹右键菜单”以及“安装与卸载”等等。

- 性能表现：有对比测试指出，在完成相同的文件拷贝任务时，Robocopy Go+的完成速度比另一款知名拷贝软件FastCopy更快。但需要注意的是，拷贝速度受硬件配置、文件数量等多种因素影响。
